# Changelog

## 0.2.0

- 読みやすさに特化したリーディングスタイルを追加。「Aa」ボタンでエディタ表示 → 明朝 → ゴシックを循環切り替えできる。
  - 明朝: 読み物向けスタイル(明朝体18px・行間2.0・840pxカラム・淡背景の引用・黒背景コードブロック)。
  - ゴシック: 技術文書向けスタイル(h1/h2の下罫線、見出し直後の詰め)。
- テーマ(ダーク/ライト)と表示スタイルの選択状態を保存し、再表示時に復元するようにした。
