# Markdown WYSIWYG

Typora風に、Markdownファイルをその場で直接WYSIWYG編集できるVS Codeのカスタムエディタです。

## 機能

- `.md`ファイルを開いた際、コマンドから見出し・太字・表・コードブロックなどがそのまま整形されて見えるWYSIWYGビューに切り替えられます。
- 編集内容は通常のテキストドキュメントと双方向に同期するため、他の拡張機能(Git、Lintなど)やVS Code標準のテキスト編集機能と共存できます。
- Milkdown(CommonMark + GFM)ベースのため、表・取り消し線・タスクリストなどGFM記法にも対応しています。

## 使い方

1. `.md`ファイルを開きます(デフォルトでは通常のテキストエディタで開きます)。
2. コマンドパレット(`Cmd/Ctrl+Shift+P`)から以下のいずれかを実行します。
   - **Markdown WYSIWYG: Markdown WYSIWYGで開く** — 現在のファイルをWYSIWYGエディタで開き直します。
   - **Markdown WYSIWYG: Markdown WYSIWYGとテキスト表示を切り替え** — WYSIWYG表示とテキスト表示を相互に切り替えます。

## 必要環境

- VS Code `1.85.0` 以上

## プライバシー

このエディタはローカルで完結して動作します。外部への通信やテレメトリの送信は一切行いません。

## 既知の制限

- 生のHTMLブロックなど、一部のMarkdown記法はWYSIWYG編集に完全対応していない場合があります。
- 非常に大きなファイルではエディタの反応が遅くなることがあります。
